# 再利用可能な作業手順（Codex Skills化候補）

## Skill 1: Recovery PRD Synthesizer
入力: 調査ノート、レビュー要約
出力: MVP要件、非機能要件、KPI
手順:
1) 不満を課題カテゴリ化
2) 成功要素を機能にマッピング
3) MVP / 非MVPを切り分け

## Skill 2: Android Blocker Feasibility Checker
入力: 機能案
出力: 実現手段比較（Accessibility/VPN/UsageStats）
手順:
1) Playポリシー観点確認
2) 技術制約・電池コスト評価
3) 段階導入案作成

## Skill 3: JP UX Copy Reviewer
入力: UI文言
出力: 日本語自然度・心理安全性レビュー
手順:
1) 機械翻訳調を検出
2) 責める表現を共感表現へ置換
3) 依存症治療断定表現を排除

## Skill 4: Privacy Permission Audit
入力: 権限一覧、データ項目
出力: 同意文言ドラフト、削除/エクスポート要件
手順:
1) 最小権限化
2) 目的外利用チェック
3) Data Safety整合性確認
