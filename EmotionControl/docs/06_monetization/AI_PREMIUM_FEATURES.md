# AI Premium Features

## 位置づけ

AI分析はPlus機能とする。AIは医療的診断、治療、依存症判定ではなく、ユーザーが入力した記録をもとにした振り返り支援として扱う。

AIの提案は参考情報であり、最終判断はユーザー自身が選ぶ。AIを使わない場合でも、基本カウンター、Daily Pledge、Daily Review、基本SOS、relapse記録、カレンダーは利用できる。

## Plusで提供するAI機能

### AI週次レポート

- 1週間の継続状況、Daily Pledge、Daily Review、SOS利用、relapse記録を整理する。
- よく出た気分、トリガー、崩れやすい時間帯をまとめる。
- 次の1週間で試せる行動を3つ以内で提案する。

### AI月次レポート

- 月単位で継続しやすいパターンと崩れやすいパターンを整理する。
- relapseの頻度、時間帯、トリガー、SOS利用との関係を振り返る。
- 大きな断定ではなく、記録から見える可能性として提示する。

### relapse傾向分析

- relapse日時。
- relapse前の気分。
- トリガー。
- 睡眠、疲労、ストレスの自己申告。
- 直前のSOS利用有無。

### 時間帯分析

- 衝動が強くなりやすい時間帯。
- relapseが記録されやすい曜日・時間帯。
- Daily Reviewで注意が必要と書かれた時間帯。

### トリガー分析

- よく選ばれるトリガー。
- トリガーと時間帯の組み合わせ。
- トリガー別の回避アクション。

### SOS利用履歴分析

- SOSを開いたタイミング。
- どのアクションを選んだか。
- SOS後にrelapseを回避できた可能性があるタイミング。
- 次回使いやすいSOSテンプレート。

### 次週の行動アドバイス

- 生活リズムを整えるヒント。
- 注意すべき時間帯。
- トリガー別の回避アクション。
- 具体的で小さく試せる行動。

### AIによる自分専用SOS提案

- 過去のrelapse傾向に基づく対策。
- 時間帯別の注意メッセージ。
- トリガー別の回避アクション。
- 自分専用SOSテンプレート。

### SOS後のAI振り返り

SOS完了後、ユーザーが落ち着いたタイミングで、短い振り返りを案内できる。ただしSOS中にはPlus課金誘導を出さない。文言は「今回の記録から次に試す工夫を整理できます」のようにし、「Plusなら防げます」のような恐怖訴求を禁止する。

### 次に崩れやすいタイミングの予測

予測は断定しない。表現は「記録上、この時間帯は衝動が強くなりやすい可能性があります」とする。必ず医療的診断ではないことを明記する。

## 分析に使うデータ

- 継続日数。
- relapse日時。
- relapse前の気分。
- relapseのトリガー。
- SOS使用履歴。
- Daily Pledge。
- Daily Review。
- 睡眠、疲労、ストレスの自己申告。
- メモ。ただしユーザーがAI分析に含める設定を有効にした場合のみ。

## 分析に使わないデータ

- 実名。
- 連絡先。
- 位置情報。
- 写真、動画。
- 他アプリの内容。
- 医療歴。
- 決済情報の詳細。
- ユーザーがAI分析対象から外したメモ、relapse記録、SOSメモ。

## AIが出してはいけない表現

- ポルノ依存が治ります。
- 脳が回復します。
- 医学的に改善します。
- あなたは依存症です。
- この方法で必ず克服できます。
- 治療効果があります。
- 再発を防げます。

## AIが使うべき表現

- 記録から見える傾向です。
- 次の一週間で試せる工夫です。
- 衝動が強くなりやすいタイミングの可能性があります。
- 生活リズムを整えるヒントとして活用してください。
- 医療的な診断ではありません。

## 出力ガードレール

- 医療、治療、診断、依存症判定を行わない。
- 恥、恐怖、断罪を強めない。
- 断定よりも「可能性」「傾向」「試せる工夫」として表現する。
- 危機的な内容が入力された場合は、AI助言で抱え込まず、専門家や緊急相談先への導線を優先する。

## AI Provider Review Notes

- 外部AIに送る場合は、送信データ、保持、ログ、学習利用の有無をプライバシー文書に明記する。
- OpenAI API等を使う場合は、利用時点の公式データ利用ポリシーを確認し、ユーザーデータがモデル学習に使われるか、保持されるログやメタデータがあるかを説明する。
- プロンプト、レスポンス、要約、特徴量、キャッシュも削除対象に含める。
