# Free / Plus Plan

## 方針

EmotionLeaveは、まずFree / Plusの2段階で設計する。Freeは基本的な習慣回復支援を十分に体験できるプラン、Plusは広告非表示とAIによる高度な振り返り支援を提供するプランとする。

## Free機能一覧

- 基本カウンター。
- アバター選択。
- ホーム画面のアバター表示。
- 称号達成。
- レベルアップモーダル。
- 軽いジャンプ、光、紙吹雪の演出。
- Daily Pledge。
- Daily Review。
- 基本SOS。
- relapse記録。
- カレンダー。
- 広告表示。

## Freeで提供する基本SOS

- ホームからSOSへ1タップ。
- 10秒アクション。
- 60秒アクション。
- 3分アクション。
- 登録した「取り戻したいもの」の再表示。
- 簡単なメモ。
- あとで振り返る。

基本SOSはEmotionLeaveの中核価値であり、無料ユーザーにも提供する。衝動が来た瞬間に支える体験は課金の前提にしない。

## Plus機能一覧

- 広告非表示。
- AI週次レポート。
- AI月次レポート。
- relapse傾向分析。
- 衝動が強くなりやすい時間帯の分析。
- トリガー分析。
- SOS利用履歴の分析。
- 次週の行動アドバイス。
- 自分専用の振り返りコメント。
- 詳細インサイト。
- AIによる自分専用SOS提案。
- 過去のrelapse傾向に基づく対策。
- 時間帯別の注意メッセージ。
- トリガー別の回避アクション。
- SOS後のAI振り返り。
- 次に崩れやすいタイミングの予測。
- 自分専用SOSテンプレート。

## FreeとPlusの違い

| 項目 | Free | Plus |
| --- | --- | --- |
| 基本カウンター | 利用可 | 利用可 |
| Daily Pledge / Daily Review | 利用可 | 利用可 |
| 基本SOS | 利用可 | 利用可 |
| relapse記録 | 利用可 | 利用可 |
| カレンダー | 利用可 | 利用可 |
| 広告 | 表示あり | 非表示 |

## 価格案

- 月額: 300〜500円程度。
- 年額: 2,900〜4,800円程度。

AI APIコストがあるため、AI分析を提供する場合は安すぎる価格にしない。MVPではFree / Plusを基本とし、Proは将来検討に留める。

## Plusにする理由

Plusの課金価値は、記録から「自分の崩れ方」を見える化し、次に試せる具体的な工夫を提示することにある。ユーザーはSOSそのものに課金するのではなく、AIによる高度な分析、個別化、次回対策に課金する。

Plusの名前は、将来的には `AI Plus`、`追加サポート`、`高度な振り返り` のように、ステータス差や上下関係を感じにくい表現を優先する。

## 無料ユーザーの体験を壊さない方針

- 初回起動直後に課金誘導を出さない。
- relapse直後に課金誘導を出さない。
- SOS中に課金誘導を出さない。
- 基本SOSを無料で提供する。
- 称号やレベルアップ達成を課金で分断しない。
- データ削除、エクスポート、プライバシー設定を課金対象にしない。

## サブスクリプション表示で明記すること

- 価格。
- 請求周期。
- 自動更新の有無。
- 無料トライアルがある場合の終了日と終了後の課金。
- 解約方法。
- 返金の扱い。
- Freeで残る機能。
- Plusで追加される機能。
- AI分析に同意しない場合でも使える機能。
