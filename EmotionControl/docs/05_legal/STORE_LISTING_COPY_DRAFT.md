# Store Listing Copy Draft

## Purpose

App Store / Google Play 申請時に使う説明文、審査メモ、データ安全性説明のたたき台。EmotionLeaveは医療、治療、診断アプリではなく、習慣回復と衝動コントロールのための記録・振り返り支援アプリとして表現する。

## Product Positioning

EmotionLeaveは、続けたい習慣を守るために、毎日の誓約、振り返り、SOS、再開記録をローカルで管理するアプリ。

アプリの価値は、ユーザーを責めることではなく、揺れた瞬間に立ち止まり、次の一手へ戻るための記録環境を提供することにある。

## Short Description

毎日の誓約、振り返り、SOSで、習慣の再開を支える記録アプリ。

## Full Description

EmotionLeaveは、衝動に流されそうな瞬間や、習慣を立て直したい時に使える記録・振り返りアプリです。

毎日の誓約、夜の振り返り、SOSアクション、再開記録を通じて、自分の状態を落ち着いて見返せます。アカウントなしで始められ、基本データは端末内に保存されます。

主な機能:

- 継続日数のカウンター
- Daily Pledge
- Daily Review
- 基本SOS
- relapse / 再開記録
- 記録一覧
- ローカルデータのエクスポート
- ローカルデータの削除
- 任意の日次リマインダー

このアプリは、医療的な診断、治療、依存症判定を行うものではありません。入力された記録をもとに、日々の振り返りを支援するためのものです。深刻な不安、自傷のおそれ、生活に大きな支障がある場合は、医療機関、専門家、地域の相談窓口に相談してください。

## Keywords

習慣, 記録, 振り返り, SOS, セルフケア, 禁欲, 衝動, 継続, 再開, 日記

## App Review Notes

EmotionLeaveは、医療的な診断、治療、依存症判定を提供しません。アプリ内のSOSは緊急医療サービスではなく、ユーザーが短時間の行動やメモで衝動から距離を取るためのセルフ記録機能です。

現行MVPではアカウント作成を必須にせず、主要データは端末内に保存されます。ユーザーは設定画面からローカルデータをエクスポートまたは削除できます。

将来Plus機能としてAI分析を提供する場合も、分析は記録の振り返り支援に限定し、医療的診断や治療として表示しません。AI分析前には同意画面を表示し、メモやrelapse記録を分析対象に含めるか選択できる設計とします。

広告を導入する場合、Freeユーザーの対象画面にバナー広告を表示します。ただし、SOS画面とレベルアップ演出中には広告を表示しません。Plusユーザーには広告を表示しません。

## Data Safety Draft

### Collected Data

現行MVPでアカウントを作らない場合、基本データは端末内に保存され、開発者サーバーへ自動送信しない。

端末内に保存する可能性があるデータ:

- 開始日
- 取り戻したいもの
- Daily Pledge
- Daily Review
- SOS履歴
- relapse / 再開記録
- 任意メモ
- 通知ON/OFFなどの設定

### Shared Data

現行MVPでは、ユーザーが明示的にエクスポート共有を選んだ場合を除き、記録データを第三者へ共有しない。

将来、広告SDK、課金SDK、AI分析、クラッシュレポート、匿名分析を導入する場合は、導入前に以下を更新する。

- プライバシーポリシー
- ストアのデータ安全性表示
- アプリ内同意画面
- データ削除手順

### Sensitive Data Notes

メモやrelapse記録にはセンシティブな内容が含まれる可能性があるため、広告SDK、分析SDK、AIプロバイダーへ送るデータは最小化する。AI分析に使う場合は、ユーザー同意、対象データ選択、削除導線を必須にする。

## Subscription Copy

Plusは、広告非表示とAIによる詳細分析・傾向分析・個別アドバイスを提供する有料プランです。

推奨表示:

- 広告なしで使えます。
- 記録から見える傾向を整理します。
- 次の一週間で試せる工夫を提案します。
- AI分析は医療的な診断や治療ではありません。
- 基本SOS、Daily Pledge、Daily Review、再開記録はFreeでも利用できます。

避ける表現:

- Plusなら必ず防げます。
- 依存症が治ります。
- 脳が回復します。
- 医学的に改善します。
- AIがあなたを治療します。

## Notification Copy

通知はセンシティブな内容を含めず、第三者に見られても意味が限定される文言にする。

推奨:

- 今日のチェックインをしませんか
- 今日の記録を残せます
- ひとことだけ振り返れます

避ける:

- 禁欲に失敗しそうですか
- relapseを防ぎましょう
- 衝動を止めましょう
- 依存症を克服しましょう

## Screenshot Guidance

- 実在の個人情報や具体的なセンシティブメモを使わない。
- SOS画面に広告や課金誘導がないことが分かるスクリーンショットを含める。
- Homeの統計、Daily Pledge、Daily Review、設定のデータ削除導線を含める。
- AI Plusが未実装の場合、ストアスクリーンショットで実装済み機能のように見せない。
