# User Personas

## Persona 1: 軽く始めたい初心者

- 名前: 佐藤 翔太
- 年齢: 24
- 状況: 夜更かしやSNSの流れで見てしまうことが増え、集中時間を取り戻したい。
- 期待: 怖いアプリではなく、軽く記録して様子を見たい。
- 不安: 初回から課金や権限を求められると閉じる。通知を見られるのも嫌。
- 成功体験: 3分以内で始められ、翌朝に「今日だけ整える」と思える。
- 必要UX: 低摩擦オンボーディング、恥ずかしくない通知、シンプルなHome、SOS。

## Persona 2: 何度も挫折している本気ユーザー

- 名前: 高橋 健
- 年齢: 32
- 状況: 何度もカウンターを始めては途切れ、途切れた瞬間にアプリを消してきた。
- 期待: 続け直せる仕組みと、衝動のピークを越える導線がほしい。
- 不安: relapseで全部ゼロに見えると、自分がダメだと感じる。
- 成功体験: 最長記録や累計成功時間が残り、「戻っても材料になる」と思える。
- 必要UX: relapse recovery、トリガー分析、最長記録/累計表示、将来ブロッカー接続。

## Persona 3: 人に見られたくないプライバシー重視ユーザー

- 名前: 山本 亮
- 年齢: 28
- 状況: 家族や職場の人にスマホ画面を見られる機会がある。
- 期待: アプリ名、通知、ロック画面で目的が露出しないこと。
- 不安: 記録、メモ、ブロック履歴、エクスポートファイルが漏れること。
- 成功体験: アプリロックと控えめな通知で、日常の自己管理ツールとして使える。
- 必要UX: local-first、アプリロック、通知文言、エクスポート警告、削除。

## Secondary Persona: ブロッカーを将来使いたい慎重派

- 名前: 中村 悠
- 年齢: 35
- 状況: 強い制限も必要だと感じるが、誤検知や解除不能が怖い。
- 期待: まず記録で信頼し、必要なら段階的に制限を強めたい。
- 不安: AccessibilityやVPNを入れた瞬間、端末が不便になること。
- 必要UX: Future Blocker Placeholder、権限説明、理由タグ、一時許可。
