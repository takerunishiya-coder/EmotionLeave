# Daily Loop

## 設計原則

- 毎日10〜30秒でも成立する。
- quick path: 10〜30秒。
- standard path: 1分以内。
- detail path: 任意で2分以内。
- 朝は「今日の軸」、夜は「明日の材料」。
- missed check-in は責めず、記録や継続を無効にしない。
- 完璧な継続だけでなく、チェックイン回数、最長記録、累計成功時間も肯定する。

## 朝の誓約

目的: 今日の回避プランを1つ置く。

標準入力:
- 固定誓約文: `今日は、衝動が来ても一度立ち止まります。`
- 回避プラン: `場所を変える`, `水を飲む`, `スマホを置く`, `SOSを開く`, `寝る前に通知`
- 気分: 任意
- メモ: 任意

完了条件:
- 固定文のみでも保存できる。
- quick pathは `固定文を確認` -> `回避プラン1つ` -> `保存` の3タップ以内。
- 選んだ回避プランはHome/SOS/Reviewで `今日の準備` として再表示される。

## 夜の振り返り

目的: トリガーと対処を明日の材料にする。

標準入力:
- 衝動の有無
- 衝動の強さ 0〜5
- トリガータグ: 疲れ、孤独、SNS、夜更かし、ストレス、退屈、一人の時間
- できた対処: 呼吸、移動、水、メモ、睡眠、ブロックで止まった
- 明日の一手
- メモ: 任意

完了条件:
- 10〜30秒の簡易保存ができる。
- 詳細入力は任意。

Quick review:

- `落ち着いて過ごせた`
- `揺れた`
- `立て直した`
- 任意でトリガー1つ

## 日中のSOS

衝動時はDaily Reviewへ誘導しない。まずSOSで30秒呼吸、場所移動、短いメモに逃がす。振り返りは落ち着いた後に任意で出す。

## 通知文言

ロック画面で見られても目的が露出しない文言にする。

- 朝: `今日のチェックインをしませんか`
- 昼: `ひと息入れる時間です`
- 夜: `今日の記録を残せます`
- 未完了: `あとで短く記録できます`
- 再開促し: `またここから始められます`

避ける:
- `禁欲の時間です`
- `ポルノを見ないで`
- `再発していませんか？`

## 継続を重くしすぎない設計

- 日次記録を逃しても記録は途切れない。
- 現行記録、最長記録、累計成功時間を並べる。
- `連続日数がすべて` という表示にしない。
- バッジは道徳評価でなく行動評価にする: `3回振り返り`, `SOSを使えた`, `再開できた`

## 状態

| 状態 | Home表示 | CTA |
|---|---|---|
| 朝未完了 | `今日の約束を置けます` | `誓約する` |
| 朝完了/夜未完了 | `今日の軸を置きました` | `振り返る` |
| 夜完了 | `今日の記録を残せました` | `記録を見る` |
| missed | `今からでも大丈夫です` | `短く記録する` |
| relapse後 | `次の24時間を整えましょう` | `再誓約する` |
