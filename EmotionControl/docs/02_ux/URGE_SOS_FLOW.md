# Urge SOS Flow

## 目的

衝動が来た瞬間に、ユーザーが考え込まずに開ける緊急導線。教育や反省ではなく、数十秒の距離を作る。

## Entry Points

- Bottom Navigation `SOS`
- Homeの大きな `SOS`
- Daily Review内の `今すぐ整える`
- 将来ブロッカー画面の `SOSを開く`

## First Screen

見出し:

`今は判断を急がなくて大丈夫です`

主CTA:

`30秒呼吸を始める`

副CTA:

- `場所を変える`
- `水を飲む`
- `スマホを置く`
- `短くメモする`
- `あとで振り返る`

競合のホームに見られる多ボタン型の便利さは参考にするが、SOSでは選択肢を増やしすぎない。高衝動時は `30秒呼吸`, `場所を変える`, `スマホを置く` を最上位に固定する。

## 10秒導線

目的: 画面を開いた瞬間に自動で距離を作る。

- 表示: `10秒だけ、ここで止まりましょう`
- 行動: 画面中央にカウント、下部に `スマホを裏返す`
- 入力: なし
- 完了後: `次に何をしますか？` と60秒導線へ

## 60秒導線

目的: 体の状態を変える。

- 呼吸: `吸う 4秒 / 止める 2秒 / 吐く 6秒`
- 行動カード: `立って別の場所へ行く`, `冷たい水を飲む`
- 完了後: `少し距離を作れました`

## 3分導線

目的: 衝動の波が落ちるまで別行動に移す。

- `5分だけ別の作業をする`
- `理由を読み返す`
- `今の気持ちを一言で書く`
- `明日の自分にメモを残す`

## Reason Re-display

Home/Goal Setupで設定した理由を短く表示:

- `取り戻したいもの: 集中`
- `今日の理由: 夜を落ち着いて終える`

文言は押し付けない:

`思い出したい時だけ、ここに置いておきます。`

目標日数だけではなく、取り戻したい生活価値を表示する。例: `集中`, `睡眠`, `時間`, `自信`, `生活リズム`。

## Memo

- placeholder: `今の気持ちをそのまま書く`
- 保存は任意。
- 後でDaily Reviewに引き継げる。
- 高衝動時にメモ必須にしない。

## Later Reflection

SOS完了後に軽い選択肢を出す:

- `ホームへ戻る`
- `あとで振り返る`
- `今日の振り返りに残す`
- `リラプスを記録する`

## Future Blocker Connection

MVPでは権限要求しない。将来はSOSから次の導線を置く。

- `15分だけルールを強める`
- `よく見る時間帯のブロックを提案`
- `ブロック理由を確認`

注意:

- ブロッカーを罰として見せない。
- `監視` ではなく `あなたが決めたルールの補助` と説明する。

## Community Handling

外部チャットや公開コミュニティは、孤独感を減らす価値がある一方で、トリガー、比較疲れ、未成年配慮、モデレーション不能のリスクがある。MVPのSOSでは外部コミュニティへ直接送らず、まず個人内で完結する短時間アクションを提供する。

## Acceptance Criteria

- Homeから1タップでSOSを開ける。
- オフラインで動く。
- 最初のアクションに入力必須がない。
- 30秒呼吸、場所移動、メモ、あとで振り返る導線がある。
- 危機語彙入力時は通常コピーで励まさず、ヘルプ導線を表示する。
