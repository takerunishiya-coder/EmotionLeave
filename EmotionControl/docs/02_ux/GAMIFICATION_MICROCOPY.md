# Gamification Microcopy

## Tone

ゲーミフィケーション文言は、軽く祝うが、煽らない。日数や称号は採点ではなく、積み重ねの記録として扱う。

使う語彙:

- 達成
- 手に入れました
- 記録
- 整える
- 立ち止まれた
- また始められた
- ここまでの記録
- できたこと
- 記録バッジ

避ける語彙:

- 失敗
- レベルダウン
- 称号剥奪
- 負け
- やり直し
- 完治
- 治療効果
- 最強
- 禁欲王

## Level-Up Copy

日数称号:

- `7日分の記録が残りました`
- `14日分の記録が残りました`
- `30日分の記録が残りました`
- `新しい記録バッジを手に入れました`
- `ここまでの記録は残っています`

行動称号:

- `立ち止まれた`
- `SOSを使えました`
- `振り返れました`
- `また始められました`
- `今日の一手を残せました`
- `立て直しの記録を残せました`
- `次の一手を選べました`

## Avatar Copy

- `一緒に進むアイコンを選びましょう`
- `あとから変更できます`
- `人に見られても分かりにくい、控えめな表示です`
- `このアイコンで始める`
- `あとで選ぶ`
- `アバターを変更`

## Relapse-Safe Copy

relapse後:

- `ここまでの称号は残っています。`
- `次の24時間をまた整えましょう。`
- `今日からの記録も、これまでの記録も、どちらも残ります。`
- `称号は消えません。ここからまた始められます。`
- `再開できたことを、静かに記録しました。`

避ける:

- `称号を失いました`
- `レベルが下がりました`
- `やり直しです`
- `連続記録が消えました`

## Notification Copy

称号/記録バッジ解放を通知する場合はMVPでは原則通知しない。将来通知する場合もロック画面で目的が露出しない文言にする。

候補:

- `新しい記録があります`
- `あとで確認できます`
- `今日の記録を見返せます`

避ける:

- `禁欲7日達成！`
- `リラプス後も称号は残ります`
- `衝動に勝ちました`

## Prohibited And Alternatives

| 禁止表現 | 代替表現 |
|---|---|
| `失敗しました` | `戻った日として記録しました` |
| `リセット` | `新しい記録を始める` |
| `レベルダウン` | `ここまでの称号は残っています` |
| `称号を失いました` | `獲得済みの称号は残ります` |
| `我慢できた` | `立ち止まれた` |
| `衝動に勝った` | `距離を作れた` |
| `依存症が改善` | `記録を続けられた` |
| `治療効果があります` | `自己管理を支援します` |
| `ランクアップ` | `記録が増えました` |
| `称号を獲得` | `記録バッジを手に入れました` |

## Store / Public Copy Notes

- アバターや称号は `楽しく続けるための小さな記録` と説明する。
- 医療、治療、改善効果を示す表現は使わない。
- ガチャ、ランダム報酬、課金による優位性がないことを明示できる。
- 素材は自作または利用許可済みである前提を開発/審査資料に残す。
