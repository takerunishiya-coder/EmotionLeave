# Microcopy Guide

## Tone

EmotionLeaveの文言は、責めない、断定しない、でも行動は具体的にする。

基本語彙:

- 整える
- 立ち止まる
- 次の24時間
- 記録する
- 振り返る
- 戻った日
- 再スタート
- 材料

## Onboarding

| 場面 | 文言 |
|---|---|
| 見出し | `今日から、戻りにくい環境を少しずつ作ります` |
| 説明 | `記録、振り返り、必要なときのSOSで、あなたの選択を支えます。` |
| CTA | `3分で始める` |
| 目標設定 | `まず、減らしたい行動を選びましょう` |
| 補足 | `あとから変更できます。今の状態に近いものを選んでください。` |
| 権限 | `ブロッカーは、必要になった時に後から設定できます。` |
| 完了 | `準備できました。まずは今日だけ整えましょう。` |

## Daily Copy

Morning:

- `今日の約束`
- `今日は、衝動が来ても一度立ち止まります。`
- `危なくなったら、まず何をしますか？`
- `今日の軸を置きました。`

Evening:

- `今日の振り返り`
- `今日は強い衝動がありましたか？`
- `きっかけに近いものを選んでください。`
- `役に立った行動はありましたか？`
- `記録しました。続いた日も、揺れた日も材料になります。`

## SOS Copy

- `今は判断を急がなくて大丈夫です。`
- `30秒だけ、ここで止まりましょう。`
- `吸う 4秒 / 止める 2秒 / 吐く 6秒`
- `少し距離を作れました。`
- `今の気持ちをそのまま書く`
- `あとで振り返れます。今はここまでで大丈夫です。`

## Relapse Copy

- `起きたことを、責めずに記録します。`
- `これは採点ではありません。次の24時間を整えるためのメモです。`
- `最長記録は残っています。`
- `ここから新しい記録を始めます。`
- `ここまで続けたことも、今日から始めることも、どちらも事実です。`
- `またここから始められます。`
- `戻った日も、次に気づくための記録です。`
- `今日は立て直しの日として残せます。`

## Calendar / Badge Copy

競合の称号名、序列、配色、アイコン配置はコピーしない。EmotionLeaveでは日別状態とバッジを採点ではなく「行動の記録」として扱う。

日別状態:

- `整った日`
- `揺れた日`
- `立て直した日`
- `記録した日`
- `SOSを使えた日`

行動バッジ:

- `3回振り返れた`
- `SOSを使えた`
- `再開できた`
- `理由を見返せた`
- `メモを残せた`

避ける:

- `成功日`
- `失敗日`
- `称号を獲得`
- `ランクアップ`
- 階級や動物名など、競合の称号体系を連想させる表現

## Notifications

ロック画面で意味が露出しない表現を使う。

| 用途 | 文言 |
|---|---|
| 朝 | `今日のチェックインをしませんか` |
| 昼 | `ひと息入れる時間です` |
| 夜 | `今日の記録を残せます` |
| SOS後 | `さっきの状態を振り返れます` |
| 権限オフ | `一部のサポートが止まっています` |
| ブロッカー停止 | `設定の確認が必要です` |
| 継続 | `今日も記録を更新できます` |
| 再開 | `またここから始められます` |

通知に出さない:

- `禁欲`
- `ポルノ`
- `衝動`
- `再発`
- `リラプス`
- `戻った日`
- ストリーク日数やトリガー名

## Privacy Copy

- `データはこの端末に保存されます。`
- `エクスポートを選んだ時だけ、JSON/CSVファイルを作成します。`
- `作成したファイルはアプリのロック対象外です。`
- `通知には詳しい内容を表示しません。`
- `表示名は任意です。実名である必要はありません。`
- `広告はMVPでは表示しません。記録中やSOS中に集中を遮りません。`

## Legal/Policy Safe Copy

- `このアプリは自己管理と習慣改善を支援するためのものです。`
- `診断、治療、医療上の助言を行うものではありません。`
- `医療上の不安がある場合は、専門家に相談してください。`
- `強い不安や自分を傷つけたい気持ちがある場合は、地域の緊急窓口や信頼できる人に連絡してください。`

Store/Aboutで併記する:

- `医療機器ではありません。`
- `病気や依存症の診断、治療、予防を目的とするものではありません。`
- `必要に応じて医療・心理・法律などの専門家に相談してください。`

## Prohibited Expressions

| 種類 | 禁止例 |
|---|---|
| 断罪 | `失敗`, `負けた`, `だらしない`, `意志が弱い`, `またやった` |
| 脅し | `このままだと人生が終わる`, `取り返しがつかない` |
| 医療断定 | `治る`, `完治`, `治療効果`, `医学的に証明`, `依存症を治す` |
| 医療化 | `治療`, `診断`, `予防`, `依存症回復`, `臨床的改善`, `医学的効果` |
| 成功保証 | `必ずやめられる`, `二度と戻りません` |
| 羞恥 | `恥ずかしい行動`, `最低な習慣`, `汚い欲求` |
| 道徳化 | `罪を断つ`, `清くなる`, `堕落を防ぐ` |
| 監視感 | `すべて見ています`, `違反を検出しました`, `あなたを監視します` |
| 競合模倣 | 競合の称号名、ボタン配置、広告前提の導線、外部チャット名の直接利用 |

## Terms

- `リラプス` は専門的に聞こえるため、主要UIでは `戻った日` または `リラプス（戻った日）` と併記する。
- `禁欲` は設定や説明では使えるが、通知やアプリアイコン周辺では避ける。
- `依存症` は医療文脈に見えやすいため、ストア文言では `習慣改善`, `自己管理`, `衝動への対処` を優先する。
- `リセット` はユーザー向け主要CTAでは避ける。必要な技術説明では `現行記録を更新`、`戻った日を記録`、`新しい記録を始める` に置き換える。
