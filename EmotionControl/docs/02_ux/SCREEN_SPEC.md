# Screen Spec

## Splash
- 画面目的: 起動とロック状態の判定。
- 表示項目: アプリ名、控えめなロゴ、必要時ロック解除。
- 主CTA: なし。
- 副CTA: なし。
- 空状態: なし。
- エラー状態: ロック解除できない時は再試行。
- 注意すべきUX: ロック画面で目的が露出しない。
- 必要なマイクロコピー: `EmotionLeave`

## Onboarding
- 画面目的: 3分以内で安心して始める。
- 表示項目: 価値説明、local-first、課金/広告なし、後で設定できること。
- 主CTA: `3分で始める`
- 副CTA: `あとで読む`
- 空状態: なし。
- エラー状態: 保存できない時は再試行。
- 注意すべきUX: 診断や医療効果のように見せない。
- 必要なマイクロコピー: `今日から、戻りにくい環境を少しずつ作ります`

## Goal Setup
- 画面目的: 取り戻したい生活に目標を接続する。
- 表示項目: 減らしたい行動、取り戻したいもの、開始日。
- 主CTA: `次へ`
- 副CTA: `あとで設定`
- 空状態: 未選択でも後で設定可能。
- エラー状態: 日付不正。
- 注意すべきUX: 詳細な性的内容や実名を聞かない。
- 必要なマイクロコピー: `今の状態に近いものを選んでください。あとから変更できます。`

## Home
- 画面目的: 今日の状態と最短アクションを示す。
- 表示項目: 現行記録、最長記録、累計成功時間、再開回数、今日の誓約/振り返り状態、SOS、将来ブロッカー状態。
- 主CTA: 時間帯に応じて `今日の誓約` または `今日の振り返り`
- 副CTA: `SOS`, `リラプスを記録`, `設定`
- 空状態: `まずは今日の誓約から始めましょう。1分で終わります。`
- エラー状態: 記録を読み込めない、権限停止。
- 注意すべきUX: 現行記録だけを大きくしすぎない。
- 必要なマイクロコピー: `ここまで続けたことは残っています`

## Daily Pledge
- 画面目的: 朝の軸を作る。
- 表示項目: 誓約文、回避プラン、気分、任意メモ。
- 主CTA: `保存してホームへ`
- 副CTA: `今日はスキップ`
- 空状態: 固定文のみで保存可能。
- エラー状態: 保存できない。
- 注意すべきUX: 1分以内、3タップ以内で完了。
- 必要なマイクロコピー: `今日は、衝動が来ても一度立ち止まります。`

## Daily Review
- 画面目的: 一日の材料を残す。
- 表示項目: 衝動有無、強さ、トリガー、対処、明日の一手、任意メモ。
- 主CTA: `記録する`
- 副CTA: `あとで`
- 空状態: `今日は落ち着いて過ごせました` で簡易保存可能。
- エラー状態: 保存できない。
- 注意すべきUX: 宿題感を出さない。relapse申告時は柔らかく専用フローへ。
- 必要なマイクロコピー: `続いた日も、揺れた日も材料になります。`

## Calendar
- 画面目的: 日別の状態を一覧する。
- 表示項目: 月表示、誓約/振り返り/relapse/SOSの軽い記号。
- 主CTA: `日付を見る`
- 副CTA: `今日へ`
- 空状態: `記録が増えると、流れが見えてきます。`
- エラー状態: 読み込めない。
- 注意すべきUX: relapse日を赤く罰する表現にしない。
- 必要なマイクロコピー: `戻った日も、次に気づくための記録です。`

## Insights
- 画面目的: 目的と傾向を思い出す。
- 表示項目: 累計成功時間、よくあるトリガー、よく効いた対処、チェックイン回数。
- 主CTA: `次の一手を選ぶ`
- 副CTA: `記録を見る`
- 空状態: `数日記録すると、傾向が表示されます。`
- エラー状態: 集計できない。
- 注意すべきUX: 医学的効果や脳科学断定を出さない。
- 必要なマイクロコピー: `数字は採点ではなく、次に気づくための材料です。`

## SOS
- 画面目的: 衝動ピークをやり過ごす。
- 表示項目: 30秒呼吸、場所移動、水を飲む、スマホを置く、短いメモ、あとで振り返る。
- 主CTA: `30秒呼吸を始める`
- 副CTA: `場所を変える`, `短くメモする`, `リラプスを記録`
- 空状態: なし。
- エラー状態: タイマー中断。
- 注意すべきUX: 高衝動時に必須入力を置かない。
- 必要なマイクロコピー: `今は判断を急がなくて大丈夫です。`

## Relapse Log
- 画面目的: 起きたことを責めずに記録する。
- 表示項目: 時刻、気持ち、トリガー、任意メモ。
- 主CTA: `次へ`
- 副CTA: `今はスキップ`
- 空状態: 選択のみで進める。
- エラー状態: 保存できない。
- 注意すべきUX: `失敗` や `リセット` を使わない。
- 必要なマイクロコピー: `これは採点ではありません。次の24時間を整えるためのメモです。`

## Restart Flow
- 画面目的: 次の24時間に戻す。
- 表示項目: 最長記録、累計成功時間、再開回数、次の一手、再誓約。
- 主CTA: `再誓約する`
- 副CTA: `ホームへ`
- 空状態: なし。
- エラー状態: 記録を更新できない。
- 注意すべきUX: 現行記録の変更と保存済み実績を分けて見せる。
- 必要なマイクロコピー: `ここまで続けたことも、今日から始めることも、どちらも事実です。`

## Settings
- 画面目的: 信頼、調整、データ管理。
- 表示項目: プロフィール、通知、アプリロック、データ、将来ブロッカー、ヘルプ。
- 主CTA: なし。
- 副CTA: 各設定行。
- 空状態: なし。
- エラー状態: 設定を保存できない。
- 注意すべきUX: 危険操作は確認を挟む。
- 必要なマイクロコピー: `データはこの端末に保存されます。`

## Privacy Lock
- 画面目的: アプリ内の機微情報を守る。
- 表示項目: ロックON/OFF、生体認証/パスコード、復帰時ロック、アプリ切替ぼかし。
- 主CTA: `ロックを有効にする`
- 副CTA: `後で`
- 空状態: 未設定。
- エラー状態: 端末ロック未設定。
- 注意すべきUX: ロック未設定でもアプリ利用は可能。
- 必要なマイクロコピー: `人に見られたくない記録を、開く時だけ守れます。`

## Data Export/Delete
- 画面目的: データの持ち出しと削除をユーザーが制御する。
- 表示項目: エクスポート範囲、形式、削除範囲、注意。
- 主CTA: `エクスポートする`
- 副CTA: `すべてのデータを削除`
- 空状態: `まだエクスポートできる記録がありません。`
- エラー状態: ファイルを作成できない、削除できない。
- 注意すべきUX: エクスポート後はアプリ保護外になると説明。
- 必要なマイクロコピー: `作成したファイルはアプリのロック対象外です。`

## Future Blocker Placeholder
- 画面目的: 将来のブロッカーへ自然につなげる。
- 表示項目: できる予定のこと、必要になる可能性のある権限、MVPでは未使用の説明。
- 主CTA: `仕組みを見る`
- 副CTA: `今は使わない`
- 空状態: なし。
- エラー状態: なし。
- 注意すべきUX: 権限許可ボタンを急がない。
- 必要なマイクロコピー: `必要になった時だけ、理由を説明して設定します。`
